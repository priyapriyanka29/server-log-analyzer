# 📊 Server Log Analyzer

## 📌 Overview

A Python tool that analyzes server log files and provides insights on errors and activity.

## 🚀 Features

* Detects errors and warnings
* Counts log entries
* Generates simple reports

## 🛠️ Tech Used

* Python

## ▶️ Run

```bash
python main.py
```

## ⚠️ Note

Supports basic log formats.
